# BCC 322
## Software Engineering I
Development of a C++ framework for the construction of simulations based on system dynamics.

### Compiling and running the functional tests
```
make all_funcional
make run_funcional
```

### Compiling and running the unitary tests
```
make all_unit
make run_unit
```
